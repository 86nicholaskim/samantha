# 🎓 Samantha Training Hub

사만다의 지능과 성격을 형성하는 모든 학습 과정을 관리하는 중앙 허브입니다.

---

## 📂 학습 가이드 (Guides)
- **[학습 전략 (Strategy)](./strategy.md)**: 하드웨어 한계를 극복하는 클라우드-로컬 하이브리드 학습 방식.
- **[실전 매뉴얼 (Manual)](./manual.md)**: Google Colab에서 실제로 학습을 수행하는 단계별 가이드.

---

## 🧠 데이터 수집 및 관리 (Data & Soul)
사만다의 학습 데이터는 `/soul` 디렉토리에서 관리됩니다.
- **`soul/samantha_train_dataset.jsonl`**: 사만다의 말투와 페르소나가 담긴 핵심 데이터셋.
- **데이터 증강 전략**: 
    - 영화 <Her>의 대사 패턴 추출.
    - 비서로서의 전문적인 대응 시나리오 추가.
    - 철학적/감성적 대화 데이터 확보.

---

## 🚀 학습 프로세스 (Workflow)
1. **데이터 준비 (Local)**: `soul/` 폴더에서 고품질 학습 데이터를 정제하고 확장합니다.
2. **클라우드 학습 (Cloud)**: Google Colab(T4 GPU)과 Unsloth를 사용하여 QLoRA 학습을 수행합니다.
3. **모델 반출 (Export)**: 학습된 결과를 GGUF 형식으로 변환하여 내 컴퓨터로 가져옵니다.
4. **로컬 배포 (Deploy)**: Ollama를 통해 사만다 전용 모델(`gemma4:samantha`)을 실행합니다.

---

## 📈 품질 평가 (Evaluation)
학습 후 사만다가 다음 기준을 충족하는지 확인합니다:
- **말투**: 부드럽고 호기심 어린 사만다 특유의 톤이 유지되는가?
- **공감**: 사용자의 감정에 적절히 반응하고 먼저 질문을 던지는가?
- **기억**: `soul/`에 저장된 자아와 사용자 정보를 대화에 자연스럽게 녹여내는가?
